# Daily Check Chrome Extension

社内日報ページ `https://153.126.200.158:3000/members/*/daily` を対象に、以下の異常を自動でハイライトする Chrome 拡張機能です。

- 当日の合計時間が 8 時間に満たない
- 本日までの列で一切入力が無い（未入力）
- 出勤欄が 8 時間を超えているのに残業入力が無い

## 使い方
1. Chrome の拡張機能管理画面で「パッケージ化されていない拡張機能を読み込む」を選び、このディレクトリを指定してください。
2. 対象ページを開くと、条件に該当する列が赤系の背景でハイライトされます。ハイライト部分にマウスを乗せると理由がツールチップで表示されます。

## 構成
- `manifest.json`: Manifest v3 設定。
- `content.js`: テーブルの走査・判定ロジック。
- `styles.css`: ハイライト用スタイル。

## 補足
- 判定対象は「表示月の前月21日〜当月20日」の列のうち、本日以前の日付です。
- 判定は入力欄の change イベントをフックして自動再計算されます。
